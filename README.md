getting started

requires python 3.9

```
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update
sudo apt install python3.9 python3.9-venv python3.9-dev
python3.9 -m venv venv
. ./venv/bin/activate
pip3 install -r requirements.txt
vim .env
export $(cat .env)
```
